# FreshMart Retail & Inventory Management Portal

A full-stack Business Intelligence (BI) and Web Development project that transforms raw retail data into an actionable executive portal. This project uses **IBM Cognos Analytics** for deep-dive data exploration and a custom **HTML/CSS/JS** web application to operationalize insights for store managers.

## 🚀 Project Overview
FreshMart operates 35 outlets across India. This portal addresses the critical business challenge of inconsistent stock levels (overstocking vs. frequent stockouts) and identifies performance gaps across different outlet sizes.

## 🛠️ Tech Stack
* **Analytics Engine:** IBM Cognos Analytics (Reporting, Exploration, Dashboards, Stories)
* **Frontend:** HTML5, CSS3 (Modern Glassmorphism UI), JavaScript (ES6+)
* **Data Source:** FreshMart Supermarket Dataset (Sales & Inventory)

## 📊 Business Intelligence (Cognos) Highlights
The project follows a structured analytical workflow:
1. **Sales Summary Reporting:** Detailed profitability analysis by Category and Sub-Category.
2. **Slow-Moving Analysis:** Identification of "dead stock" using conditional threshold styling.
3. **Trend Visualization:** Interactive weekly sales trend monitoring.
4. **AI Exploration:** Discovery that Small Outlets yield higher margins than Large Outlets.
5. **Executive Storytelling:** A 6-scene Quarterly Business Review for stakeholders.

## 🌐 Web Portal Features
The developed web portal is architected directly upon the **FreshMart dataset**, ensuring 1:1 data parity:
* **Role-Based Authentication:** Secure entry for Store Managers, Supervisors, and Viewers.
* **Dynamic KPI Tiles:** All metrics, including **Stock Value** and **SKU counts**, dynamically reflect the distribution of the 14 localized product categories.
* **Smart Alert System:** 'Low Stock' notifications are programmatically integrated with the **Reorder Level** and **Stock Closing** attributes.
* **Operational Readiness:** Empowers Store Managers to resolve real-time inventory anomalies identified during the initial **IBM Cognos Exploration** phase.

## 📈 Key Findings
* **Total Revenue:** ₹12.9M with a Profit of ₹2.32M.
* **Stockout Impact:** 968 cumulative days out of stock were identified as the primary revenue leak.
* **Operational Insight:** Small outlets outperform larger counterparts in efficiency; strategy should focus on replicating this model.
